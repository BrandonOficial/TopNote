Como Usar

Click

Clique nas colunas para criar um bloco de notas móvel.

Mover

Segure o botão esquerdo do mouse para mover o bloco de notas entre as colunas.

Click Duplo

Dê um duplo clique com o botão esquerdo do mouse para editar os textos dentro do bloco de notas. Para excluir o bloco, basta apagar o texto que está nele.
